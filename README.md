# ionic_sim

## 개발 환경

- node(npm) version: v12.16.0(6.13.4)
- ionic version: 6.1.0
- cordova version: 9.0.0

## plugin 설치

`ionic cordova plugin add cordova-plugin-sim`

`npm install @ionic-native/sim`
